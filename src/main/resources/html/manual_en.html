User Manual · Hogwarts

Welcome! This manual guides you step by step to use Hogwarts Manager without technical jargon. It's designed so anyone can follow it easily, even if they haven't used computer magic before. We hope you enjoy it!

Basic system usage

What can you do here?


  • View students of a house or all of Hogwarts.
  • Add a new student (make sure to warn them if a troll is in the bathroom).
  • Edit data directly in the table.
  • Delete a student if you made a mistake… or if they wandered into the Forbidden Forest. Aragog might have a feast.
  • Undo the last changes.

Getting started (2 minutes)


  1. Open the application. (This step is easier than seeing Snape smile, don’t complain.)
  2. Choose Hogwarts to see everything, or a specific house to work only with it.
  3. Explore the student table: you can sort it or edit cells.

Tip: If any word seems strange, don't worry. Follow the icons and steps. And take off your goggles, there are no mandrakes here.

Help and troubleshooting

Quick overview of the application


House selectorDropdown at the top: Hogwarts, Gryffindor, Hufflepuff, Ravenclaw, Slytherin.
FormFields: First Name, Last Name, Year, Patronus. (House fills automatically.)
ButtonsAdd, Delete, Undo.
TableStudent list: ID, First Name, Last Name, House, Year, Patronus. Edit by double-clicking.
MenuAt the top: File, Help and Languages.

Frequently asked questions


  • I don’t see my house → make sure it’s selected in the dropdown.
  • Can’t add a student → check all fields are filled and Year is a number.
  • Edited a cell but it doesn’t change → press Enter after typing. Error messages will appear if something goes wrong.
